Cette page permet d'importer les propositions du CFP dans Trello sous forme de cartes dans plusieurs tableaux (un tableau Trello par type de propositions).
Il suffit d'indiquer le nom de l'organisation à laquelle rattacher les tableaux et de cliquer sur le bouton "Import".

Attention ! L'organisation est le nom affiché dans l'URL lorsque l'on clique sur l'organisation. Il diffère du nom affiché dans Trello.
